Um site moderno, responsivo e cinematográfico para apresentar as maiores estreias do cinema. Desenvolvido com HTML5, CSS3 e JavaScript puro.

## 🌐 Demonstração
- **Site online:** [https://SEU-USUARIO.github.io/cineestreia/](https://SEU-USUARIO.github.io/cineestreia/)
- **Repositório:** [https://github.com/SEU-USUARIO/cineestreia](https://github.com/SEU-USUARIO/cineestreia)

## ✨ Funcionalidades

- 🎨 Design clean e cinematográfico com paleta preto / vermelho escuro / dourado / cinza escuro
- 📱 Totalmente **responsivo** (mobile, tablet e desktop)
- 🎬 Vitrine com 3 filmes em destaque: **Homem-Aranha**, **Avatar** e **De Volta para o Futuro**
- 🖱️ Efeitos de hover com **zoom 3D**, sombra dourada e animação de inclinação
- ⏳ Animação de **loader** ao carregar a página
- 🔢 **Contador animado** de filmes em cartaz
- 📩 Formulário de contato funcional com feedback de sucesso
- 📄 Páginas individuais com sinopse completa para cada filme (abrem em nova aba)
- 🎭 Menu mobile responsivo

## 🚀 Como Publicar no GitHub Pages

1. **Criar repositório público no GitHub**
   - Nome sugerido: `cineestreia`
   - Marque "Public" e crie.

2. **Enviar arquivos**
   Via linha de comando:
   ```bash
   cd cineestreia
   git init
   git add .
   git commit -m "🎬 Primeiro commit - CineEstreia"
   git branch -M main
   git remote add origin https://github.com/SEU-USUARIO/cineestreia.git
   git push -u origin main
   ```
   
   Ou simplesmente arraste os arquivos no site do GitHub (botão **Add file → Upload files**).

3. **Ativar GitHub Pages**
   - Vá em **Settings** → **Pages**
   - Em *Source*, selecione a branch **main** e pasta **/ (root)**
   - Clique em **Save**
   - Aguarde 1-2 minutos e acesse: `https://SEU-USUARIO.github.io/cineestreia/`
